## Web Application

Look at the [React documentation](https://reactjs.org/) to learn more.

### Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

### Development Server

Start the development server on http://localhost:3000

```bash
npm start
```

### Production

Build the application for production:

```bash
npm run build
```
